# ETU003250-ETU003274-Web-Design

Bienvenue dans le projet de Web Design ! Ce projet contient une application divisée en deux parties : un Front-office pour les utilisateurs et un Back-office pour l'administration.

## 🚀 Démarrage rapide

Pour lancer le projet, assurez-vous d'avoir Docker installé, puis exécutez la commande suivante à la racine du projet :

```bash
docker compose up -d
```

## 🌐 Liens d'accès

Une fois les conteneurs démarrés, vous pouvez accéder aux interfaces via les adresses suivantes :

- **Front-office** : [http://localhost:8080](http://localhost:8080)
- **Back-office** : [http://localhost:8081](http://localhost:8081)
